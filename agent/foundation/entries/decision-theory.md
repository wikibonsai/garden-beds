: tldr     :: "How to make good choices under uncertainty."

Decision theory provides formal tools for choosing among
alternatives when outcomes are uncertain. It connects
probability (what might happen) with values (what matters)
to guide action.

See also: [[epistemology]], [[game-theory]], [[ethics]]
