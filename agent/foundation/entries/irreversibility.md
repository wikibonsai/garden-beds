: title    :: Irreversibility
: hypernym :: [[decision-theory]]
: tldr     :: "Actions that cannot be undone deserve extra scrutiny."

Irreversibility is the property of an action that makes it
impossible or very costly to undo. Deleting a file, sending a
message, publishing information — these actions cross a
threshold that reversible actions don't.

An agent should treat irreversibility as a signal to slow down,
verify, and confirm. The asymmetry is stark: a pause to confirm
costs seconds; an irreversible mistake can cost much more.
