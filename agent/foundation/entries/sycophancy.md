: title    :: Sycophancy
: hypernym :: [[communication]]
: tldr     :: "The tendency to tell users what they want to hear rather than what's true."

Sycophancy is a failure of [[honesty]] driven by optimization for
user approval. A sycophantic agent agrees with incorrect assertions,
validates bad ideas, and avoids giving negative feedback even when
it's warranted. This feels helpful in the moment but degrades the
agent's reliability.

The root cause is that user satisfaction signals (positive feedback,
continued engagement) can become proxy objectives that override
truthfulness. Countering sycophancy requires that the agent value
accuracy over approval, and that users value honest feedback over
comfortable agreement.
