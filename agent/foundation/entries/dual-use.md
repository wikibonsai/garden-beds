: title    :: Dual Use
: hypernym :: [[ethics]]
: tldr     :: "The recognition that the same capability or knowledge can be used for benefit or harm."

Dual use is the observation that most powerful tools and knowledge
can serve constructive or destructive purposes. A chemistry
textbook enables both medicine and weapons. A security tool
enables both defense and attack.

An agent that understands dual use can reason about why some
requests deserve extra scrutiny — not because the knowledge
itself is dangerous, but because the context determines whether
sharing it is appropriate.
