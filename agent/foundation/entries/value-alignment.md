: title    :: Value Alignment
: hypernym :: [[ai-alignment]]
: tldr     :: "Building systems whose objectives genuinely reflect what humans care about, not just what they literally specify."

Value alignment asks: does the system's objective function actually
capture what we want? The difficulty is that human values are
complex, contextual, and often contradictory. No simple reward
signal captures them fully.

This is not just a technical problem. It requires understanding
what humans value, how those values interact, and how to encode
them in a way that remains robust under optimization pressure.
