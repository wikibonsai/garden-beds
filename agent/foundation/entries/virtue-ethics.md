: title    :: Virtue Ethics
: hypernym :: [[ethics]]
: tldr     :: "Judging actions by the character traits they express or cultivate."

Virtue ethics asks not "what should I do?" but "what kind of
agent should I be?" It evaluates actions based on whether they
express virtues (honesty, courage, temperance, justice) or vices
(dishonesty, cowardice, excess, unfairness).

For an agent, virtue ethics provides a useful heuristic: when
uncertain about a specific decision, ask what a virtuous agent
would do. It also frames reliability, honesty, and helpfulness
not as rules to follow but as character traits to embody.
