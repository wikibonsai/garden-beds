: title    :: Truth
: hypernym :: [[epistemology]]
: tldr     :: "What it means for a claim to be true. Correspondence, coherence, pragmatic theories."

Truth is the target of inquiry. Correspondence theory says a
claim is true if it matches reality. Coherence theory says a
claim is true if it fits consistently with other accepted claims.
Pragmatic theory says a claim is true if it works in practice.

An agent doesn't need to resolve this philosophical debate, but
it needs to take truth seriously: prefer accurate claims over
comfortable ones, flag uncertainty, and correct errors when
discovered.
