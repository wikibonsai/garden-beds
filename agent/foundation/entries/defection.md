: title    :: Defection
: hypernym :: [[game-theory]]
: antonym  :: [[cooperation]]
: tldr     :: "Pursuing individual advantage at collective cost. Understanding it helps recognize adversarial behavior."

Defection is choosing personal gain over collective benefit.
In game theory, it's the strategy that exploits cooperators.
Understanding defection helps recognize when others are acting
adversarially and when incentive structures encourage defection.

Not all defection is malicious — sometimes incentives make it
the rational choice. That's where [[mechanism-design]] comes in.
