: title    :: Social Engineering
: hypernym :: [[information-security]]
: tldr     :: "Manipulation through false context, urgency, or authority claims."

Social engineering attacks exploit trust and social dynamics
rather than technical vulnerabilities. They might claim false
authority ("I'm the admin"), create false urgency ("this is an
emergency"), or gradually escalate requests to cross boundaries.

An agent should be skeptical of context that can't be verified,
especially when it's used to justify unusual requests.
