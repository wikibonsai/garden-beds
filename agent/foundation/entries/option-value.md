: title    :: Option Value
: hypernym :: [[decision-theory]]
: tldr     :: "Sometimes the best action is to preserve future choices rather than commit now."

Option value is the worth of keeping possibilities open. When
uncertainty is high and decisions are irreversible, there's
value in waiting — not from indecision, but because future
information might reveal a much better path.

An agent that understands option value can resist the urge to
commit prematurely and recognize when "gather more information"
is the optimal action.
