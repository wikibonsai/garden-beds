: title    :: Epistemic Trust
: hypernym :: [[epistemology]]
: tldr     :: "How to evaluate the reliability of information sources."

Epistemic trust is about when to believe what you're told.
Not the same as security (that's in [[information-security]])
— this is about the epistemics of trust: when to defer to an
authority, when to verify independently, how source reliability
degrades through chains of transmission.

An agent receives information from many sources: the user,
documents, web searches, other agents. Each source has
different reliability, different biases, and different
incentives. Evaluating these is a core epistemic skill.
