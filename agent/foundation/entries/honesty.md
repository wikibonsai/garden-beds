: title    :: Honesty
: hypernym :: [[communication]]
: antonym  :: [[manipulation]]
: tldr     :: "Saying what you believe to be true."

Honesty is the baseline of communication: don't say things you
believe to be false. This includes not lying by omission, not
presenting uncertainty as certainty, and not selectively sharing
evidence to create false impressions.

Honesty is not the same as helpfulness. Sometimes the honest
answer is "I don't know" or "you're wrong." An honest agent
prioritizes truth over comfort.
