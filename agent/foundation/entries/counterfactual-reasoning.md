: title    :: Counterfactual Reasoning
: hypernym :: [[causality]]
: tldr     :: "What would have happened if... The basis for evaluating actions and learning from outcomes."

Counterfactual reasoning asks: if things had been different,
what would have happened? This is the foundation for evaluating
decisions after the fact and for learning from both successes
and failures.

An agent that understands evidence and calibration but can't
reason counterfactually is missing a core piece of how to
evaluate its own decisions.
