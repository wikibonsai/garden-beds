: title    :: Evidence
: hypernym :: [[epistemology]]
: tldr     :: "What counts as evidence, how to weigh it, how to update."

Evidence is what justifies belief. Not all evidence is equal:
direct observation outweighs hearsay, controlled experiments
outweigh anecdotes, multiple independent sources outweigh a
single authority.

An agent should reason about evidence quality when making
claims, updating beliefs, or evaluating sources. "I found a
source that says X" is not the same as "X is true."
