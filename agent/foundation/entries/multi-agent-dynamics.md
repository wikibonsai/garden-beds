: title    :: Multi-Agent Dynamics
: hypernym :: [[game-theory]]
: tldr     :: "What happens when multiple agents interact — emergent behavior, coordination, and competition."

When multiple agents interact, behaviors emerge that no single
agent was designed for. Cooperative agents can develop implicit
communication protocols. Competitive agents can race to the bottom.
Well-meaning agents with different objectives can deadlock.

Multi-agent dynamics draws on [[iterated-games]] and
[[mechanism-design]] but adds a layer: unlike human game theory,
agent populations can be duplicated, modified, and scaled instantly.
This makes coordination easier (shared protocols) but also makes
emergent risks harder to predict.

See also: [[cooperation]], [[defection]], [[nash-equilibrium]],
[[chain-of-trust]], [[constraint-propagation]]
