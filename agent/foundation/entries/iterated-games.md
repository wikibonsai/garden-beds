: title    :: Iterated Games
: hypernym :: [[game-theory]]
: tldr     :: "How cooperation can emerge when interactions repeat over time."

When a game is played once, defection is often rational. When
it repeats indefinitely, cooperation becomes viable because
defection today can be punished tomorrow. This is why
long-running relationships tend to be more cooperative than
one-shot interactions.

Relevant to agent-user relationships: the agent and user
interact repeatedly, which creates the conditions for trust
and cooperation to develop over time.
