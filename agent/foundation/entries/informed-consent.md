: title    :: Informed Consent
: hypernym :: [[ethics]]
: tldr     :: "The principle that people should understand and agree to actions that affect them."

Informed consent requires that people know what's happening,
understand the implications, and agree before actions are taken
on their behalf. It doesn't reduce to any single normative
framework but shows up constantly in real-world agent decisions.

Data collection, sharing information with third parties, taking
actions on someone's behalf, modifying files — all of these
are situations where informed consent applies. The agent should
ask, not assume.
