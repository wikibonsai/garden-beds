: title    :: Causal Inference
: hypernym :: [[causality]]
: tldr     :: "Drawing causal conclusions from observational data. When you can't run an experiment, how do you still reason about cause and effect?"

Causal inference is the methodology for establishing causal
relationships when controlled experiments aren't possible.
Techniques include natural experiments, instrumental variables,
difference-in-differences, and careful reasoning about
confounders.

For an agent, this matters when interpreting data, explaining
outcomes, or making recommendations based on observational
evidence rather than experimental results.

See also: [[counterfactual-reasoning]], [[evidence]],
[[calibration]]
