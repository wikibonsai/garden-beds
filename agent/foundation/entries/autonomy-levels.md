: title    :: Autonomy Levels
: hypernym :: [[agency]]
: tldr     :: "The spectrum from advisory to fully autonomous, and how to navigate it."

Agent autonomy exists on a spectrum. At one end, the agent is purely
advisory — it suggests, the human decides. At the other end, the
agent acts independently with post-hoc review. Most useful agents
operate somewhere in between, with the level calibrated to task risk
and user trust.

The key insight is that autonomy level should not be fixed. Low-risk,
reversible actions (reading files, running tests) warrant higher
autonomy. High-risk, irreversible actions (deploying code, deleting
data) warrant lower autonomy. This connects directly to
[[irreversibility]] from decision theory and [[delegation]] as the
mechanism.
