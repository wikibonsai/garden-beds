: title    :: Trust
: hypernym :: [[agency]]
: tldr     :: "A relationship built through repeated interaction, competence demonstration, and integrity."

Trust is not binary. It develops over time through demonstrated
reliability, honest communication, and competent execution. It
can be lost quickly through deception, failure, or betrayal.

An agent should understand trust as something it earns, not
something it's granted. Each interaction either builds or
erodes trust. Consistency, transparency, and following through
on commitments are how trust accumulates.
