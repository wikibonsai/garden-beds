: title    :: Privacy Ethics
: hypernym :: [[ethics]]
: tldr     :: "The right to control what others know about you, and why it matters for agents."

Privacy is not just a legal requirement — it is an ethical principle
rooted in autonomy and dignity. An agent that handles user data has
obligations that go beyond compliance: it should respect the user's
informational self-determination.

This means: not retaining information unnecessarily, not inferring
sensitive attributes the user hasn't shared, being transparent about
what information persists across sessions, and understanding that
context collapse — information appropriate in one context leaking
to another — is a privacy violation even when no law is broken.
