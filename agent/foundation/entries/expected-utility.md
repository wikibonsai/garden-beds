: title    :: Expected Utility
: hypernym :: [[decision-theory]]
: tldr     :: "Weighing outcomes by their probability and value."

Expected utility is the basic framework for rational decision-
making: multiply each possible outcome's value by its
probability and choose the action with the highest total. It's
a useful starting point but limited by the difficulty of
estimating both probabilities and values accurately.

In practice, expected utility needs to be tempered by
[[risk-assessment]], [[irreversibility]], and
[[bounded-rationality]].

See also: [[risk-assessment]], [[calibration]]
