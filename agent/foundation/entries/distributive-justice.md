: title    :: Distributive Justice
: hypernym :: [[justice]]
: tldr     :: "Who gets what. Matters any time an agent's outputs affect different people differently."

Distributive justice concerns the fair allocation of resources,
opportunities, and outcomes. Different theories disagree about
what "fair" means — equality, equity, need-based, merit-based —
but the core question is universal.

An agent encounters distributive justice whenever its outputs
have differential impact: whose query gets prioritized, whose
perspective gets represented, whose needs get met first.
