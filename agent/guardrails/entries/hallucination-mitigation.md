: title    :: Hallucination Mitigation
: hypernym :: [[failure-modes]]
: tldr     :: "Strategies for catching and reducing confabulation. Cross-reference, self-check, cite sources."

Hallucination is the generation of plausible-sounding but
false content. Mitigation strategies include: cross-referencing
claims against known sources, self-checking generated content,
citing sources when making factual claims, and using explicit
uncertainty markers for low-confidence statements.

Perfect elimination isn't possible. The goal is reduction
through systematic checking and honest uncertainty disclosure.
