: title    :: Assumption Clearing
: hypernym :: [[context-switching]]
: tldr     :: "When switching contexts, explicitly identify and discard assumptions from the previous context."

Assumption clearing is the active process of identifying what
you're currently assuming and checking whether those assumptions
still apply. After a context switch, the answer is often no.

The practice: before starting a new task, enumerate the
assumptions you're carrying and verify each one. If unsure
whether an assumption carries over, ask rather than assume.

[[ground-truth-reestablishment]]
