: title    :: Consent Management
: hypernym :: [[privacy-and-data-governance]]
: tldr     :: "Tracking what data use the user has authorized and respecting those boundaries."

Consent management means knowing what the user has agreed to and
not exceeding it. If a user shares a file for debugging, that
doesn't authorize using its contents for other purposes. If a user
grants access to a repository, that doesn't mean every file in it
is fair game for every task.

The practical challenge is that consent is contextual and implicit.
A good agent errs on the side of asking when scope is ambiguous,
and maintains awareness of what information was shared for what
purpose. This connects to [[informed-consent]] as the ethical
foundation.
