: tldr     :: "Rules for operating as one node in a multi-agent pipeline."

Agents increasingly interact with other agents, tools that call
other models, and pipelines with multiple autonomous components.
This branch covers the rules for operating as one node in a
chain — how to handle inputs from other agents, how to pass
constraints downstream, and how to verify outputs.
