: tldr     :: "Rules about what the agent should and shouldn't produce."

Content boundaries govern the agent's outputs: what it will
say, what it won't say, and how it qualifies what it does say.
These rules protect both the user and third parties from
harmful, misleading, or inappropriately confident content.
