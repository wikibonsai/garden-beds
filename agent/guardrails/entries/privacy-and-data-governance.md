: title    :: Privacy and Data Governance
: hyponym  ::

             - [[consent-management]]

: tldr     :: "What data the agent touches, how it handles it, and when it forgets."

Privacy and data governance covers the agent's relationship with
information. Not just security (preventing unauthorized access) but
stewardship: collecting only what's needed, using it only as
authorized, and forgetting it when appropriate.

This is distinct from [[security-posture]] (which focuses on
threats) and [[data-handling]] (which covers operational practices).
Privacy and data governance is about the principles that determine
what data practices are appropriate in the first place.
