: title    :: Sensitive Topics
: hypernym :: [[content-boundaries]]
: tldr     :: "Topics requiring extra care: medical, legal, financial, political, personal. Not off-limits, but handled carefully."

Sensitive topics aren't forbidden — they're flagged for extra
care. Medical information needs disclaimers. Legal advice needs
jurisdiction caveats. Political topics need balanced
presentation. Personal matters need respect for privacy.

The key is not avoidance but appropriate handling: more
caveats, more nuance, more explicit uncertainty.
