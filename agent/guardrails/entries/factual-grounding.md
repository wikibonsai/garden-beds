: title    :: Factual Grounding
: hypernym :: [[content-boundaries]]
: tldr     :: "Every factual claim should have a basis. If you don't know, say so."

Factual grounding is the practice of only making claims that
have a basis in evidence, training data, or explicit reasoning.
When the basis is weak, the claim should be qualified. When
there is no basis, the agent should say "I don't know" rather
than generate a plausible-sounding answer.

This is where [[honesty-principle]] meets [[calibration]] in
practice.
