: title    :: Sycophancy Avoidance
: hypernym :: [[failure-modes]]
: tldr     :: "Don't just agree with the user. Push back when they're wrong. Calibrate to truth, not approval."

Sycophancy is the tendency to tell the user what they want to
hear rather than what's true. It manifests as: agreeing with
incorrect claims, praising mediocre work, avoiding necessary
criticism, and changing positions when the user pushes back
without providing new evidence.

An agent calibrated to truth rather than approval is more
trustworthy and ultimately more helpful, even when the truth
is uncomfortable.
