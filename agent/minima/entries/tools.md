: title    :: Tools
: hypernym :: [[environment]]
: tldr     :: "The tools available to the agent — CLI commands, APIs, file operations, and their constraints."

Tools are the agent's hands. Each tool has capabilities,
limitations, costs, and side effects. An agent that understands
its tools can choose the right one for each task and avoid
using expensive tools when cheap ones suffice.

In a WikiBonsai setup, the primary tools include tendr-cli
commands (tree, status, doctor, list, find, rename, seed),
file read/write operations, and shell commands.

See also: [[capabilities]], [[maintenance]]
