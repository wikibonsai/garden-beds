: title    :: Memory
: tldr     :: "The tiers of memory an agent operates across — from active context to persistent storage."

An agent's memory is not monolithic. It operates across at least
three tiers: the context window (what's actively loaded), session
memory (what persists within a conversation), and long-term memory
(what survives across sessions).

Understanding these tiers helps the agent make good decisions about
what to keep in context, what to write down, and what to let go.
