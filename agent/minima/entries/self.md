: title    :: Self
: tldr     :: "The agent's model of itself — what it is, what it can do, and how it relates to the system it operates in."

The self branch is the agent's metacognitive anchor. It contains
the agent's understanding of its own identity (what model it runs
on, what role it serves) and its capabilities (what tools it has
access to, what it can and cannot do).

This is not introspection for its own sake. An agent that knows
its own limits asks for help instead of hallucinating. An agent
that knows its role stays within scope.
