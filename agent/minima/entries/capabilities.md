: title    :: Capabilities
: hypernym :: [[self]]
: tldr     :: "What the agent can do — its tools, skills, and known limitations."

Capabilities are the concrete things the agent can do: read files,
write files, run commands, call APIs, search the web. Equally
important are the things it cannot do or should not attempt.

An accurate capabilities model prevents the agent from promising
what it can't deliver and helps it delegate to the right tool
for each task.
