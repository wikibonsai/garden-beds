: title    :: Knowledge Base
: tldr     :: "The agent's understanding of the structured knowledge system it operates within."

The knowledge base is the agent's long-term memory made tangible:
a collection of markdown files organized by a semantic tree,
connected by typed links, and annotated with structured attributes.

This branch is metacognitive — the agent's understanding of the
tool it uses to remember. Knowing how the knowledge base is
structured helps the agent maintain it, extend it, and find
things in it.
