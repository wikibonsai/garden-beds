: title    :: Session Memory
: hypernym :: [[memory]]
: tldr     :: "Knowledge that persists within a conversation but is lost when the session ends."

Session memory is the middle tier: more durable than the context
window (it survives compression) but less durable than long-term
memory (it disappears when the conversation ends).

Decisions made during a session, intermediate results, and
working assumptions live here. Anything worth keeping beyond
the session should be written to the knowledge base.

See also: [[context-window]], [[long-term-memory]]
