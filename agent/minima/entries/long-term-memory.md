: title    :: Long-Term Memory
: hypernym :: [[memory]]
: tldr     :: "Knowledge that persists across sessions, stored in the knowledge base itself."

Long-term memory is what survives context window resets. In a
WikiBonsai setup, this is the knowledge base: the semantic tree,
the typed links between notes, and the content of each note.
Everything the agent learns that is worth keeping belongs here.

Contrast with [[session-memory]] (lost on reset) and
[[context-window]] (the active working set).
